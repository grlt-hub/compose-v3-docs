## TypeScript with Tags

### TaskStatus

Represents the possible execution outcomes of a Task.

{/* prettier-ignore */}
```ts
type TaskStatus = 
  | { name: "done" } 
  | { name: "skip" } 
  | { name: "fail"; error: unknown };
```
