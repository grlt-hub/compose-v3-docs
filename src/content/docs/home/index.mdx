## Reuse code for any task

Write logic once and run it anywhere. Need to mock an API or change a config for a specific page? Just provide different values<br/>
Build features that work in any part of the app.

<Sandbox template="vanilla" code={logger} options={{ editorHeight: 440, showConsole: true }} />

## Toggle features with ease

Enable or disable any feature with a single toggle. Turning off a feature automatically stops all its dependent features.<br/>
Stop cluttering code with `if` conditions and manual cleanup.

<Sandbox template="vanilla" code={featureToggle} />

## Build with confidence

Stop checking if everything is wired correctly. Catch missing dependencies the moment your app starts.<br/>
Never debug a missing dependency in production.

<Sandbox template="vanilla" code={guard} />

## See how things connect

Stop digging through folders to understand your architecture. Get a clear map of your system.<br/>
Keep your project transparent, no matter how large it grows.

<Sandbox template="vanilla" code={graph} options={{ layout: "console", showConsoleButton: false }} />

## The way logic should be

No decorators, no hidden metadata. Just clean, scalable, and easy-to-test JavaScript.
